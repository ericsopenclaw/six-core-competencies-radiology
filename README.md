# 六大核心能力｜放射科一問一答大字報

這是一個用於錄製「期末座談會暨影片回饋」的單頁式大字報 / teleprompter 網頁。

## 內容主軸

- 放射科醫師視角
- 臨床照護情境
- 角色：主治醫師兼臨床教師 × 醫學實習生
- 形式：一問一答
- 架構：6 個章節
- 長度：約 8 分鐘
- 六大核心能力
- 去識別化、病人隱私與危急值通報

## 使用方式

- `← / →`：上一章 / 下一章
- `空白鍵`：自動捲動 / 停止
- `F`：全螢幕
- `T`：黑底 / 白底切換
- 右下角「設定」可調整字體大小與捲動速度
