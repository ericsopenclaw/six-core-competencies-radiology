# 六大核心能力｜急診中風取栓一問一答大字報

這是一個用於錄製「期末座談會暨影片回饋」的單頁式大字報 / teleprompter 網頁。

## 內容主軸

- 放射科醫師視角
- 急診中風病人照護流程
- 影像診斷與腦中風取栓
- 角色：主治醫師兼臨床教師 × 醫學實習生
- 形式：一問一答
- 架構：6 個章節
- 長度：比前版更長，約 10–12 分鐘，可依語速調整
- 對應六大核心能力：醫學知識、病人照護、溝通技巧、制度下臨床工作、專業素養、工作中學習與改進

## 使用方式

- `← / →`：上一章 / 下一章
- `空白鍵`：自動捲動 / 停止
- `F`：全螢幕
- `T`：黑底 / 白底切換
- 右下角「設定」可調整字體大小與捲動速度
